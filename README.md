# Excalidraw Clone

A self-hosted Excalidraw whiteboard built with **Next.js** and **TypeScript**, powered by the official [`@excalidraw/excalidraw`](https://www.npmjs.com/package/@excalidraw/excalidraw) package.

## Features

- Full Excalidraw experience — hand-drawn shapes, text, arrows, freehand
- Dark / light mode
- Undo / redo
- Export PNG, SVG, clipboard
- Zoom & pan
- Keyboard shortcuts
- Zero configuration

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

1. Push this repo to GitHub.
2. Go to [vercel.com](https://vercel.com) → **Add New Project** → import the repo.
3. Framework: **Next.js** (auto-detected). Root directory: `.`
4. Click **Deploy**.

## Stack

- [Next.js 16](https://nextjs.org) (App Router)
- [TypeScript](https://www.typescriptlang.org)
- [@excalidraw/excalidraw](https://www.npmjs.com/package/@excalidraw/excalidraw)
